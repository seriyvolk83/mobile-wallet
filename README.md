# mobile-wallet

The library provides convenient API for using `vcx` library.

## Dependencies

- `vcx`, v. 0.8.2 (included into the project)

## How to use

1. Import in `Podfile`:
```
pod 'MobileWallet'
```
2. Disable Bitcode in Build Settings
3. Check `Examples/` on how to configure and use the library.
